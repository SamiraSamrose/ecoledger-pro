# User Guide

## Getting Started
1. Launch EcoLedger Pro application
2. Navigate using top tab bar
3. Access functions through respective interfaces

## Loan Origination
1. Click Loan Origination tab
2. Fill borrower information form
3. Enter financial and ESG metrics
4. Submit for AI scoring
5. Review approval decision

## Document Upload
1. Click Document Vault tab
2. Enter loan ID
3. Select document type
4. Upload file
5. Review OCR results

## Portfolio Trading
1. Click Trading Floor tab
2. Create portfolio or view listings
3. Execute trades with buyer ID
4. Monitor transaction in ledger

## Monitoring
1. Click Covenant Monitor tab
2. Enter loan ID
3. Review compliance status
4. Check alerts for violations

## Rate Adjustments
1. Click Rate Engine tab
2. Enter loan ID
3. Calculate adjustment
4. Review savings summary
